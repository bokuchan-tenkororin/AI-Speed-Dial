「AIスピードダイアル」は、ChatGPT、Gemini、Claudeなどの主要なAIチャットのウェブサイトがあらかじめ登録されているChromeの拡張機能です。ホームボタンや新規タブから呼び出すことができます。  
  
・AIスピードダイアルのインストール  
(1)Chromeの拡張機能の「デベロッパーモード」をONに設定し、  
(2)「パッケージ化されていない拡張機能を読み込む」をクリックし、  
(3)この拡張機能が保存されているフォルダを選択する。  
  
◆ ホームボタンでAI Speed Dialが表示されない場合  
１．設定＞拡張機能＞拡張機能を管理＞AI Speed Dialの詳細のIDをコピーする。  
２．設定＞デザイン＞ホームボタンを表示する新しいタブページ＞カスタムのウェブアドレスを入力に以下のアドレスを入力する。  
chrome-extension://xxx/newtab.html  
xxx は、コピーしたAI Speed Dialの詳細のID
