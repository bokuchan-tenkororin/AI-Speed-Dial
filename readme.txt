「AIスピードダイアル」は、ChatGPT、Gemini、ClaudeなどのAIチャットのウェブサイトがあらかじめ登録されているChromeの拡張機能です。ホームボタンや新規タブの追加ボタンから呼び出すことができます。  
  
【AIスピードダイアルのインストール】  
(1) Chromeの拡張機能の「デベロッパーモード」をONに設定し、  
(2) 「パッケージ化されていない拡張機能を読み込む」をクリックし、  
(3) この拡張機能が保存されているフォルダを選択します。  
  
【ホームボタンでAI Speed Dialが表示されない場合】  
(1) 「設定」＞「拡張機能」＞「拡張機能を管理」＞「AI Speed Dial」のIDをコピーし、  
(2) 「設定」＞「デザイン」＞「ホームボタンを表示する新しいタブページ」＞「カスタムのウェブアドレスを入力」に以下のアドレスを入力します。  
chrome-extension://xxx/newtab.html  
xxx は、コピーしたAI Speed Dialの詳細のID  
  
【AIスピードダイアルの使い方】  
・ホームボタンまたは新規タブのボタンをクリックすると、AIスピードダイアルが表示されます。ホームボタンや新規タブのボタンをクリックしたときに、AIスピードダイアルが表示されるかどうかは、オプションで変更できます。  
・AIスピードダイアルには、上側にウェブ検索の入力欄が表示され、下側にデフォルトで登録されているAIチャットのアイコンが表示されます。AIチャットのアイコンは、ドラッグ＆ドロップで位置を入れ替えることができます。  
・「＋」のアイコンをクリックすると、ユーザが任意のウェブページを登録することができます。  
・ウェブ検索の入力欄にテキストを入力してEnterを押すと、デフォルトの検索エンジンのウェブページが開き、入力したテキストの検索結果が表示されます。デフォルトの検索エンジンは、オプションで変更できます。  
・AIチャットのアイコンの１つをクリックすると、クリックしたAIチャットのウェブページが開き、チャットを開始することができます。  
・ユーザが登録したウェブページのアイコンをクリックすると、そのウェブページが開き、閲覧することができます。
